API Usage Examples

Python Example

        
import requests

url = 'http://api.larssies.site/random'
response = requests.get(url)
data = response.json()

print(data)

JavaScript Example

        
fetch('http://api.larssies.site/random')
.then(response => response.json())
.then(data => {
    console.log(data);

    // Get the image URL
    const imageURL = data.url 
});

API Features

Description

A simple API that returns a random picture on a GET request.

Example Usage

GET Request: curl http://api.larssies.site/random

Response:

        
{
 "url": "http://api.larssies.site/cdn/RANDOM_PICTURE/",
 "success": "true"
}
